# GoldenLine

GoldenLine is a comprehensive project consisting of a .NET backend and a React frontend.

## Project Structure

- **GoldenLine.Desktop**: Desktop application logic.
- **GoldenLine.Launcher**: Launcher utility.
- **goldenline-frontend**: React-based web interface (Tailwind CSS, Vite).
- **GoldenLine.sln**: Visual Studio Solution file.
